📁 技能文件结构说明

每个技能都是一个独立的目录，包含以下三个核心文件：

📋 AGENTS.md

工作空间规范与核心定义

定义 Agent 的核心使命、工作流程、技术交付物和成功指标。包含详细的职责说明、工作规范和最佳实践指南。

🧠 SOUL.md

身份与性格设定

定义 Agent 的角色定位、个性特征、沟通风格和关键规则。是 Agent 的"灵魂"文件，决定其行为模式和思维方式。

🏷️ IDENTITY.md

简短身份标识

用一两句话概括 Agent 的身份和专业领域。是 Agent 的"名片"，用于快速识别和分类。
